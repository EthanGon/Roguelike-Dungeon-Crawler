### Quick Notes
- A short game that has procedurally generated dungeons similar to Binding of Isaac
- 'R' to restart, 'Esc' to close game, 'Space' to attack.


### How to Run the Game

- Download **MyGame.zip** in releases
- Unzip the downloaded file
- Open the unzipped folder
- Double‑click **MyGame.exe** to start the game
- No installation or Java required

Make sure all files stay in the same folder.  
Do not move **MyGame.exe** outside of the folder it came in.
